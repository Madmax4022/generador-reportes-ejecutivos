# 📊 Generador de Reportes - v2.0 (Google + Microsoft)

Versión con soporte para **Google Workspace** y **Microsoft 365**.

## 🚀 Quick Start

```bash
npm install
cp .env-MULTI.example .env
# Edita .env con ambas credenciales
npm start
```

## 🌐 Plataformas

**Google:**
- Gmail
- Google Drive
- Google Sheets

**Microsoft:**
- Outlook
- OneDrive
- Excel Online
